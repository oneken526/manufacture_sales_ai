見積一覧
作成済み見積の一覧を確認する画面
URL /quotations
HTTPメソッド GET
アクセス権限 sales admin
見積ステータス一覧
ステータス DB値 意味 受注確定
DRAFT 1 見積中。初期ステータス 可（在庫があれば）
CONVERTED 2 受注転換済み。すでに受注が作成された 不可
CANCELLED 3 キャンセル（現在は画面から操作不可） 不可
EXPIRED 4 期限切れ。有効期限を過ぎた見積 不可

ステータス 上記ステータス表のいずれか
